# Akses Trading ZYY
Simulasi latihan trading investasi menggunakan saldo virtual.

⚠️ **PENTING**:
- Hanya untuk tujuan pembelajaran & latihan
- Tidak menggunakan uang asli
- Bukan saran investasi resmi
- Tidak menjamin keuntungan apa pun

📊 Fitur:
- Saldo awal virtual Rp10.000.000
- Simulasi Beli/Jual Bitcoin, Saham TLKM, dan Emas
- Riwayat transaksi otomatis
